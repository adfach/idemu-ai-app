# **App Name**: IdemuAI

## Core Features:

- Prompt Creation & Management: Create, edit, save, and categorize AI prompts with a freemium model (10 free prompts).
- AI-Powered Prompt Optimization: Utilize Gemini API for prompt optimization, auto-suggestion, and tone/style enhancement.
- Bilingual Interface: Real-time bilingual interface (Bahasa Indonesia & English) with auto-detection of user language preference.
- Prompt Marketplace: Browse, search, filter, and 'remix' public prompts from the community. Users can like and favorite prompts.
- AI Mentor Chatbot: Gemini-powered chatbot provides in-app help, tutorials, and prompt improvement tips, acting as a helpful tool.
- User Authentication: Secure authentication using Google Sign-In, Email/Password, and GitHub login via Firebase Authentication.
- Payment Integration: Integrate Midtrans Payment Gateway for Indonesian users, supporting monthly & yearly plans. Payment status stored in Firestore.

## Style Guidelines:

- Primary color: Deep Indigo (#663399), reflecting the AI & creativity.
- Background color: Light Lavender (#E6E0FF), providing a gentle contrast and modern feel.
- Accent color: Electric Blue (#7DF9FF), drawing attention to key CTAs.
- Body font: 'Inter', sans-serif, modern and neutral. Headline font: 'Poppins', sans-serif, geometric, for a contemporary feel.
- Use modern, minimalist icons that complement the glassmorphism aesthetic. Icons should be intuitive and visually appealing.
- Implement a responsive design with a sidebar navigation, a main workspace for prompt creation, and a floating AI mentor chat window.
- Use subtle transitions and animations for a smooth and engaging user experience. Animated loading states and interactive elements to enhance user feedback.
